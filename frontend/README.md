# Miss Excellence — Frontend (React + Vite + Tailwind)

## Démarrage

1) Installer:

```bash
cd frontend
npm install
```

2) Variables d’environnement

Copie le fichier:

- `frontend/env.example.txt` → `frontend/.env`

Assure-toi que `VITE_API_BASE` pointe vers le backend (par défaut `http://localhost:3000`).

3) Lancer:

```bash
npm run dev
```

Frontend: `http://localhost:5173`
